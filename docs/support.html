Getting Help

  1. Check the in-app User Guide — tap Settings › About › User Guide
  2. Report a bug or request a feature — open an issue on GitHub

Common Questions

What formats are supported?

  • Books: EPUB, PDF, TXT, RTF
  • Comics: CBZ, CBR, CB7

How do I import books?

  • Files app: Tap the import button and select files
  • WiFi Transfer: Go to Settings › Import Books › WiFi Transfer
  • WebDAV / OPDS: Configure remote sources in Settings

How do I recover my purchases?

Go to Settings › Restore Purchases to recover your premium subscription or lifetime purchase.

Can I sync between devices?

Enable iCloud Sync in Settings to sync reading preferences across your devices.
